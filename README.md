# AI自動翻訳＆画像OCR LINEチャットボット (GAS版)

Google Apps Script (GAS) と LINE Messaging API を用いて開発した、サーバーレスの自動翻訳チャットボットです。
テキストの翻訳に加えて、画像内の文字を読み取って翻訳するOCR機能も実装しています。

## 💡 開発の背景とアピールポイント
単なる機械翻訳ではなく、「日常会話で実際に使える自然な表現」をシームレスに取得したかったため自作しました。
一番の工夫点は、Gemini 2.5 Flashに対して精緻なプロンプトエンジニアリングを行っている点です。

### 主な機能と技術的な工夫
- **文脈に応じた自然な翻訳**: 直訳ではなく、日常会話で使われる自然な英語のイディオムを出力するようAIに指示しています。複数の解釈ができる場合は候補を2〜3個提示するようにプロンプトを調整しました。
- **画像OCR翻訳機能の実装**: LINEで送られた画像データを `UrlFetchApp` で取得してBase64に変換し、Geminiのマルチモーダル機能に直接投げることで、「英語の看板や書類の写真を送るだけで日本語に翻訳される」という実用性の高い機能を数行のシンプルなコードで実現しました。
- **サーバーレス環境**: ランニングコストと保守の手間を削減するため、Google Apps ScriptによるWebhookで完全にサーバーレス化して稼働させています。

## ⚙️ 使用技術
- Google Apps Script (バックエンドロジック全般)
- LINE Messaging API (メッセージ送受信・画像取得)
- Gemini API (`gemini-2.5-flash` モデル / テキスト翻訳・画像文字認識)
